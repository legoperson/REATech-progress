Problem, Method, and Result

Problem: when a single test record has a large error, we need to know whether this reflects model failure or normal unit-level variation. Method: for 2-bedroom units, each observed rent is compared with the median rent for the same building, month, and bedroom group. Result: the median absolute percentage deviation is only 1.92%, and the 90th percentile is about 8.37%, meaning most within-building deviations are small.

Interpretation

  • Natural variation within the same building, month, and bedroom group is usually modest.
  • If a record differs from the building-month median by more than 20%, it is already well above typical within-building dispersion.
  • Very large single-record errors should therefore be checked for data-quality issues or model-mechanism problems, not only explained as normal unit variation.

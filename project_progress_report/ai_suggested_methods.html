Initial Screening: Methods 1-6

"AI suggested" means these six method ideas were proposed as candidate approaches for handling changing suburb relationships, outliers, and prediction stability. The initial Methods 1-6 screening was designed as a clean method comparison. Because it avoided some of the later imputation and practical fallback logic, coverage was extremely low, so these results should be read as method diagnostics rather than deployable model performance.

What Each Method Means

  • Method 1: Static + OLS. Selects correlated suburb median-rent series using all historical data, then fits ordinary least squares.
  • Method 2: Rolling Selection + OLS. Recomputes suburb correlations inside a recent rolling window, then fits ordinary least squares.
  • Method 3: Time-Decay Weighted OLS. Uses all historical data but gives more recent observations more influence through exponential time decay.
  • Method 4: Static + Robust Regression. Keeps static suburb selection but replaces OLS with Huber regression to reduce the influence of outliers.
  • Method 5: Rolling + Robust. Combines rolling suburb selection with Huber regression to address both relationship drift and outliers.
  • Method 6: Rolling + Robust + Fallback. Adds a fallback rule to Method 5 for cases where the robust prediction appears unstable.

Focused Rerun: Method 4 Static + Huber

  • Uses the imputed base-suburb space.

Focused Rerun: Method 5 Rolling 36 Months + Huber

  • Re-selects suburbs inside a rolling 36-month window.
  • Uses the imputed base-suburb space.
  • Uses lag-expanded features.
  • Regression model: HuberRegressor.

Result: n_test = 14136, n_covered = 10662, coverage_ratio = 0.754244, MAPE = 0.087149.

Takeaway

The main takeaway is that the first Methods 1-6 screening was useful for comparing model ideas, but the coverage was too low for deployment. In the more practical focused reruns, Method 4 static Huber was more stable than Method 5 rolling Huber, with higher coverage and lower MAPE.

Problem, Method, and Result

Problem: we needed a clean comparison set of base-suburb estimation methods that can be evaluated separately from the retained production-style model. Method: two robust regression variants were tested: Static + Huber and Rolling 36 Months + Huber. Result: Static Huber performed better in this experiment, with higher coverage and lower MAPE than the rolling-window Huber version.

Method 4: Static + Huber

  • Static suburb selection.
  • Uses the imputed base-suburb space.
  • Uses lag-expanded features.
  • Regression model: HuberRegressor.

Result: n_test = 14136, n_covered = 13387, coverage_ratio = 0.947015, MAPE = 0.080847.

Method 5: Rolling 36 Months + Huber

  • Re-selects suburbs inside a rolling 36-month window.
  • Uses the imputed base-suburb space.
  • Uses lag-expanded features.
  • Regression model: HuberRegressor.

Result: n_test = 14136, n_covered = 10662, coverage_ratio = 0.754244, MAPE = 0.087149.

Takeaway

The main takeaway is that the static Huber approach is currently more stable than the rolling 36-month Huber approach. In this setup, the rolling window did not improve accuracy and reduced coverage.
